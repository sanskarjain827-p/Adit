# maggieaurcode — High-Craft Design & Engineering Studio

A premium, story-driven digital systems portfolio built for **maggieaurcode** design and development studio. The website showcases custom identity-led web architectures, interactive canvas physics, and a fully polished user experience tailored for modern branding.

## 🛠️ Tech Stack & Features

- **Frontend**: React + Vite (ESModules)
- **Styling**: Modern, responsive Vanilla CSS with dynamic CSS variables
- **Animations**: GSAP (GreenSock Animation Platform) + Framer Motion
- **Performance**: High-performance HTML5 `<canvas>` rendering loop for 60fps cursor physics
- **Special Effects**: 
  - **Antigravity Particle Vortex**: Concentric rings of rotating dashes that react to cursor repulsion and swirls.
  - **Theme Toggle**: Liquid Glass transitions between light and dark modes with color-inverting custom cursors.
  - **Custom Cursor**: Desktop-only trailing spring cursor with `mix-blend-mode: difference` for optimal legibility.

## 🚀 Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sanskarjain827-p/Adit.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the local development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
